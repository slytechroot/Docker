- name your container what you want:
docker run -d -p 6001:6379 --name redis-older redis:4.0
docker ps
docker stop <container-ID>

#another docker command for debugging
docker ps
- if you want to check something in the container itself, such as check the files, etc. We need to (-it) interact with the terminal.
docker exec -it <container-name> /bin/bash
ls
pwd
cd /
exit
- you can also use the name as well. 
docker exec -it redis-older /bin/bash
